Dappy test dapp


This dapp has been built to check that dapps loaded in dappy work fine and that the dangerous behaviours are blocked by the "sandboxed" attribute on the iframe tag in which dapp code is executed.



  • Dapp local css should be evaluated, make sure the background is light grey and not white
  • Dapp local javascript should be evaluated : javascript has not been evaluated
  • Bulma css library should be loaded, make sure this button is nice :
  • React javascript library should be loaded : React is not loaded
  • Javascript alert() should be disabled: make sure you didn't see one
  • Form submission should be blocked: make sure you were not taken to w3c.org
  • Javascript should not be able to access parent context (Dapp main context): parent context is not blocked
  • Javascript popups should be disabled (window.open): make sure a tab/window was not openned
